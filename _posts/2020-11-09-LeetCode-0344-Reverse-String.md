### 1. How I sloved
&nbsp;&nbsp;&nbsp;&nbsp;I had to reverse the string. So I just used 'reverse()' to do it.

### 2. Code
```python
class Solution:
    def reverseString(self, s: List[str]) -> None:
        s.reverse()
```
